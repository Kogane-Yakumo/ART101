Lab 6 - Arrays and Objects

This lab is about the compound data types of JavaScript, more specifically arrays and objects. Arrays are lists which start at index 0 which may confuse some people. Arrays can hold data of any type even other compound data types. Objects can be assigned properties, think of these as sub-variables of a variable. We practiced creating and calling them in our console and in our documents.
